Вариант 2.

1. Написать программу которая меняет порядок букв в слове на противоположный. 
   Без использования метода reverse в StringBuilder, StringBuffer.

2. Написать программу которая будет удалять первые или последние n заданных символов (маленьких или больших) 
   в заданной строке. Пример, заданна строка “You know nothing Jon Snow”, 
   нужно удалить первые 3 символа “o” = “Yu knw nthing Jon Snow”. 

3. Написать программу, которая разбивает слово на буквы, удаляет дубликаты(регистр игнорируется) 
   и сортирует их в алфавитном порядке.
